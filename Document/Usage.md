# How to Use This Project

## Overview
This document explains how to use this project to analyze SwimTopia psych sheet data and predict which team is most likely to win based on recorded event times.

It is intended for SwimTopia administrators who have access to psych sheets and want to use existing time data to support meet outcome predictions. The documentation focuses on how to use the tool and interpret results and does not require knowledge of the underlying implementation.
